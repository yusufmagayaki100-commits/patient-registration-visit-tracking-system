# Patient Record Management System

This project is a web-based Patient Record Management System developed using PHP, MySQL, HTML, and CSS. The system helps healthcare organizations manage patient records, track patient visits, generate reports, and manage user accounts efficiently.

## Features
- User Authentication
- Patient Record Management
- Add Patient Visits
- View Visit Records
- Report Generation
- User Account Management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP
